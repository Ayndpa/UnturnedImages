# Unturned 图片库 (Unturned Images)

这是一个由社区驱动的开源项目，致力于构建最全面的 Unturned 游戏资产图片资源库。

## 项目核心价值
无论你是正在开发 Unturned 插件的开发者，还是致力于构建游戏相关周边服务的网站管理员，Unturned 图片库都能为你提供标准化的游戏内资产图标资源。通过这个统一的图片资源仓库，你无需自行截取和处理游戏内图标，即可快速构建美观、功能完善的用户界面。

## 如何接入
只需遵循简单的命名规则，即可轻松集成到你的项目中。

**使用逻辑：**
你可以通过物品 ID 或载具 ID 获取对应的资源。资源路径结构采用 ID 映射，通过组合基础路径、资产类型和资产 ID 即可获取目标图片的路径。

## 贡献与合作
作为一个开源项目，我们欢迎社区成员贡献缺失的图片资源，特别是广受欢迎的模组资产。有关具体的贡献指南，请查阅 [CONTRIBUTING.md](/CONTRIBUTING.md)。
